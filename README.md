# Smart Resume Analyzer

An AI-powered resume analysis system built using FastAPI and NLP techniques.

## Features

- Resume PDF Upload
- Resume Parsing
- Skill Extraction
- Job Description Matching
- Match Score Generation
- NLP-based Similarity Analysis

## Tech Stack

- Python
- FastAPI
- spaCy
- scikit-learn
- TF-IDF
- Cosine Similarity
- PyMuPDF / pdfplumber
